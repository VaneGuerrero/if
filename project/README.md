# ImaginaITS Clone (estructura base)

Réplica estructural y visual inicial de https://imaginaits.com/.

## Fuente original
- Web principal: https://imaginaits.com/
- Sitemap objetivo: https://www.imaginaits.com/sitemap.xml (bloqueado en este entorno; ver reporte)

## Estructura
- `pages/`: páginas clonadas
- `components/`: header/footer reutilizables
- `assets/images/`: imágenes reales extraídas + fallback manual
- `styles/`: estilos globales

## Nota
En este entorno no fue posible descargar binarios desde la web origen por bloqueo de red (HTTP 403 vía túnel). Se dejaron URLs originales y fallback explícito `IMAGEN NO DISPONIBLE – REVISIÓN MANUAL` cuando no se pudo recuperar el recurso.
